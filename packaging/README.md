# Packaging-Quellen

Dieses Verzeichnis enthaelt versioniertes Quellmaterial fuer den Build-Prozess.

## Inhalt

- `pyinstaller/minan_v1.spec`: PyInstaller-Spezifikation fuer den One-Folder-Build.
- `pyinstaller/windows_version_info.txt`: Windows-Version-Metadaten fuer die EXE.

## Abgrenzung

- Hier liegt nur Quellkonfiguration.
- Generierte Build-Artefakte landen in `build/` und `dist/` und werden nicht versioniert.
