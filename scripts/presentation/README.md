# Presentation-Skripte (intern)

Dieses Verzeichnis enthaelt interne Hilfsskripte zur Erzeugung von
Praesentationsdateien und PDF-Exports.

## Einordnung
- Nicht Teil des produktiven MinAn-Kerns
- Nicht fuer den normalen Nutzer-Startpfad erforderlich
- Duerfen Projektartefakte in `presentation/` erzeugen

## Ausfuehrung
- Ausfuehrung vom Projektordner aus empfohlen
- Pfadauflosung ist auf den Projektroot angepasst
