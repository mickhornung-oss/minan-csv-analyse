MinAn 1.4 - CSV-Schnellanalyse

Was ist MinAn?
MinAn ist ein portables Windows-Tool zur schnellen Analyse von CSV-Dateien.

So starten Sie MinAn:
1. MinAn.exe im selben Ordner starten.
2. Im Fenster auf "CSV oeffnen" klicken oder den Schnellstart aufrufen.

Eigene CSV laden:
- "CSV oeffnen" waehlen und eine Datei auswaehlen.

Beispieldatei laden:
- "Info / Schnellstart" oeffnen.
- Auf "Beispieldatei laden" klicken.

Wo landen Berichte?
- Standardmaessig in: output\reports

Wo landen CSV-Exporte?
- Standardmaessig in: output\csv

Wichtig:
- Alles funktioniert portable innerhalb dieses Ordners.
- Die Originaldatei wird nicht ueberschrieben.
